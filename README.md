# kindle_web_react

## Installation

```bash
git clone
cd kindle_web_react
npm install
npm start
open http://localhost:3000
```

## run in Local
if you run in local, please change index.html file
<script src="/dist/bundle.js"></script>
to
<script src="/static/bundle.js"></script>
